# Pico2W DualSense 5 Bridge
> 将 Pico2W 变成 DS5 手柄的无线适配器

# 功能特点
 - 支持HD震动

# 使用方法
1. 按住 Pico 上的BOOTSEL进入刷机
2. 将 .uf2 文件拖入进去
3. 将 DS5 手柄进入蓝牙配对模式
4. Enjoy it

# 当前问题:
1. 使用PS键进行重连时，会配对失败
2. 陀螺仪功能在游戏异常（但是在线检测工具正常）
